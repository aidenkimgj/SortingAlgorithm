# CPRG311-Assignment_02: Sorting Out Sorting.

## How the program does:
This application implements and perform experimental analysis on six sorting
algorithms: bubble, insertion, selection, merge, quick and shell sorts.
It sorts by three types of shape (height, volume and base area) from the largest
to smallest (descending order),

## The Authors:
Hong Sangmean, Kim Gookjin, and Sung Min-Chul

## How to run the program:
1) Run Command Prompt (CMD)
2) Change directory which includes the sort.jar file.
3) Place txt file to be sorted in the same directory
4) Typle in the command lines like the example below.

## Command examples:

`java -jar sort.jar -sB -th -fpolyfor1.txt`

`java -jar sort.jar -TA -Fpolyfor1.txt -SS`

`java -jar sort.jar -fpolyfor1.txt -Si -tV`

`java -jar sort.jar -sB -th -fpolyfor1.txt`

`java -jar sort.jar -TA -Fpolyfor1.txt -Sq`

`java -jar sort.jar -fpolyfor1.txt -SZ -tV`
